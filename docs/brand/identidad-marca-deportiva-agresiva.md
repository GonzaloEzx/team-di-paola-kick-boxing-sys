# Identidad de Marca Deportiva Agresiva

## Estado

Version 0.2 del brief de identidad visual para Team Di Paola Kick Boxing.

La direccion elegida es **Ruta B: Bestia Competitiva**. La marca debe sentirse como una academia profesional y un equipo de competencia: disciplinada, intimidante, deportiva y con presencia fuerte.

Este documento todavia no es un manual de marca final. Ordena decisiones para despues aplicar la identidad en sistema, redes, ropa, piezas graficas y posibles redisenos de logo.

## Suposicion mas peligrosa

Estamos asumiendo que `DPC` significa `Di Paola Competicion` y que funciona principalmente como marca para ropa o linea comercial.

Como no esta confirmado, `DPC` debe tratarse por ahora como una submarca o aplicacion comercial vinculada a Team Di Paola, no como reemplazo del nombre principal.

## Punto de quiebre probable

La mezcla de agresividad grunge, perro intimidante, diagonales, bloques pesados y colores oscuros puede funcionar muy bien para redes, competencia y ropa, pero puede volverse agotadora dentro del sistema administrativo.

La regla base es: **marca fuerte, interfaz usable**.

## Solucion mas simple primero

Crear una identidad madre para `Team Di Paola` y una aplicacion secundaria para `DPC`.

- `Team Di Paola`: academia profesional, equipo de competencia, comunicacion institucional y sistema.
- `DPC`: ropa, merchandising, piezas comerciales y posibles capsulas visuales mas agresivas.

Esto evita mezclar todos los mensajes en un solo logo o una sola pantalla.

## Direccion elegida: Ruta B, Bestia Competitiva

La marca toma como eje un perro intimidante, contraste alto, composiciones duras y energia competitiva.

Debe transmitir:

- Disciplina.
- Competencia.
- Respeto.
- Fuerza.
- Pertenencia.
- Profesionalismo.

No debe transmitir:

- Desorden visual.
- Violencia gratuita.
- Marca generica de gimnasio.
- Estetica infantil.
- Exceso de textura que tape informacion.

## Posicionamiento

Team Di Paola debe verse como una academia profesional y equipo de competencia.

El publico es amplio: alumnos nuevos, competidores, familias, distintas edades y compradores de indumentaria. Por eso la marca puede ser intimidante, pero no inaccesible.

Prioridad de lectura:

1. Academia profesional.
2. Equipo de competencia.
3. Comunidad deportiva.
4. Marca comercial de ropa y accesorios.

## Logo y simbolos

La direccion elegida mantiene el perro intimidante como simbolo principal.

Decisiones actuales:

- Mantener un perro como figura de identidad.
- Evitar que parezca lobo generico si el equipo lo reconoce como perro.
- Necesita funcionar como logo completo y como avatar chico.
- Necesita funcionar en ropa, idealmente bordado o estampado.
- No hay elementos historicos confirmados que no se puedan cambiar.

Riesgo:

El logo actual puede tener problemas de derechos, baja resolucion o exceso de detalle para bordado/avatar. Antes de usarlo como definitivo hay que revisar si conviene redibujarlo.

Regla sugerida:

- Logo completo: perro + `Team Di Paola`.
- Version reducida: cabeza/perfil del perro.
- Version ropa: `DPC` + perro simplificado o marca de garra/energia.

## Tono visual

Direccion elegida:

- Agresividad grunge con desgaste, pero moderna.
- Fotos editadas con contraste deportivo.
- Diagonales, cortes duros y bloques pesados.
- Amarillo en detalles, botones y titulares.
- Blanco/gris solo como texto sobre fondos oscuros.

Reglas de aplicacion:

- Textura y desgaste deben usarse como capa de atmosfera, no como fondo de lectura.
- Las diagonales deben guiar la mirada, no romper la navegacion.
- Los bloques pesados sirven para titulares, promociones y competencia.
- En pantallas del sistema, usar menos textura y mas claridad.

## Tipografia

La marca necesita tipografia pesada, condensada y deportiva.

Uso recomendado:

- Titulares: fuente condensada, fuerte, preferentemente en mayusculas.
- Texto de sistema: fuente legible, neutra y rapida de leer.
- Botones: peso alto, texto corto, buena separacion.

No todo debe ir en mayusculas. Las mayusculas funcionan para impacto, pero usadas en exceso bajan legibilidad.

## Aplicacion en el sistema

La interfaz debe tener marca deportiva, pero priorizar claridad administrativa.

Decisiones actuales:

- El sistema puede usar identidad deportiva.
- El amarillo puede usarse en botones principales.
- No se espera uso durante muchas horas seguidas, pero igual conviene evitar contraste agotador.
- Debe existir modo de impresion para recibos, listados o reportes con bajo consumo de tinta.

Reglas:

- Login y dashboard pueden tener mayor carga de marca.
- Formularios, tablas y reportes deben ser mas sobrios.
- Los botones principales pueden usar `#F6C90E`.
- Alertas no deben depender solo del amarillo de marca.
- El modo impresion debe reducir fondos oscuros y texturas.

Pendiente:

## Aplicaciones por contexto

### Competencia

Puede usar la version mas agresiva:

- Perro intimidante.
- Fondos oscuros.
- Amarillo alto contraste.
- Texturas grunge.
- Cortes diagonales.
- Titulares en mayusculas.

### Academia

Debe equilibrar fuerza y profesionalismo:

- Menos textura.
- Mas orden.
- Fotos de entrenamiento y alumnos.
- Comunicacion clara.
- Jerarquia visible de horarios, clases y profesores.

### Sistema

Debe ser funcional:

- Fondo oscuro controlado.
- Paneles claros o gris oscuro.
- Botones amarillos.
- Tablas legibles.
- Tipografia limpia.
- Sin texturas detras de datos.

### Ropa y DPC

Puede ser mas compacta y simbolica:

- Monograma `DPC`.
- Perro simplificado.
- Alto contraste.
- Menos texto.
- Composiciones pensadas para estampa o bordado.

## Deuda tecnica o de marca

No hay deuda tecnica grave si este trabajo queda en documentacion.

Si se implementa en el sistema sin definir logo final, tipografias y jerarquia `Team Di Paola` / `DPC`, se genera deuda visual: CSS dificil de ajustar, pantallas inconsistentes y riesgo de tener que rehacer login, dashboard y piezas graficas.

Riesgo de escala:

- Aplicar grunge en componentes reutilizables puede complicar mantenimiento.
- Usar una sola fuente deportiva para todo el sistema puede empeorar legibilidad.
- Mezclar DPC y Team Di Paola sin reglas puede confundir marca institucional y comercial.
